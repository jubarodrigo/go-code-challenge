## Runing Local

---

1. Install the following dependencies:
    - Choose one of the following free container management tool:
        - [Go >= 1.22](https://go.dev/doc/install/)


2. Run command on terminal to up service:

 ```bash
   $ go run main.go
   ```

3. Routes:
    - REST: http://localhost:3333
